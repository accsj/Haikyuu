# Haikyuu
Um projeto de website baseado no anime Japônes Haikyuu.

Tecnologias que planejo usar: Html + Css + Js com Node.Js

Objetivos futuros com esse projeto:

    Deixar responsivo.
    Adicionar features em js
    Criar um banco de dados e posteriormente usar Node.Js
    Criar uma aba de "comunidade", onde possam ser feitos posts por usuários.
